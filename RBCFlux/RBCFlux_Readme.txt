RBCFlux_Readme.txt

Code written by Xiang Ji, modified by Jacob Duckworth Kleinfeld lab @UCSD
Code to load and process dual-channel two photon images of pial vessels 
with Cy5.5 labeled lumen and RBCs labeled with CellTrace CFSE.

Order: Pre-processing, processing, post_processing, result_integration.
Separate files used for processing ReaChR experiment data, to analyze only 
initial portions of the data etc.
